The Evolution of Christmas Music

In this project, we will be analyzing the history and development of Christmas Music using Python! To do this, we will be using datasets from hymnary.org, a website with many records of hymnals. We will analyze the development and popularity of Christmas music throughout past 500 years, answering 6 important questions, like "how has Christmas music become more secular in the past century?" To answer these questions, we won't make broad assumptions based on personal experience, but we will find real statistics based on real data.

Question 3: How can we model the popularity of Christmas songs?

To find an appropriate model for the popularity of Christmas songs, it would be wise to first plot the number of instances for each song.

In order to plot this, we will be using Matplotlib. In the following block of code, we convert the merged list of Christmas songs into a NumPy array, and plot it as a histogram. For this example, I decided to use bins of width 2 for asthetic purposes. I also limited the x-axis to have a max of 50, preventing any outliers from making the graph to small to see.

In[]:
import matplotlib.pyplot as plt

counts = merged_list["totalInstances"].to_numpy()
bins=np.arange(min(counts), max(counts) + 2, 2)
plt.hist(counts, bins=bins)
plt.xlim((0, 50))
plt.show()

Now that we can see how frequent each "totalInstances" attribute is, it is apparent that it closely follows the Pareto distribution! This distribution is very common in economics and many other fields (see "Occurence and Applications").

After tweaking some constants to make the distribution fit our data, we can plot the Pareto distribution overtop our histogram. In this case, we can simply use Matplotlib's plot() function, passing an array of x values and an array of y values according to the distribution's formula.

In[]:

Question 6: Does the song's popularity trend match the trend of number of hymnals produced?

A simple way to find out visually is to overlay the the proportion graph on top of the graph of Hymnals produced

In[]:
plt.hist(array_to_plot[:,0], bins=bins, color='r')
proportion_scalar = 1000
plt.plot(bins[:-1], proportion_scalar * proportions, color='b')
plt.show()

This might be interesting to see, but it's not scientific! To find an answer to our question based on data, we can make a scatter plot and find the correlation between the two variables.

In[]:
plt.scatter(total_counts, proportions)
plt.show()

Then, to find the correlation between the two variables, we use the Pearson Correlation Formula.

In[]:
x_mean = np.mean(total_counts)
y_mean = np.mean(proportions)


numerator = np.sum((total_counts-x_mean)*(proportions-y_mean))

sd_x = (np.sum((total_counts-x_mean)**2))**(0.5)
sd_y = (np.sum((proportions-y_mean)**2))**(0.5)

correlation = numerator/(sd_x*sd_y)
correlation
Out[]:
0.9038533884978152

Woah! That's a very strong correlation! To finish everything off, let's fit a linear regression line to the graph.

In[]:
from sklearn.linear_model import LinearRegression

X = np.array(total_counts[:,np.newaxis])
Y = np.array(proportions)

regression_model = LinearRegression()
regression_model.fit(X, Y)

plt.plot(X, regression_model.predict(X))
